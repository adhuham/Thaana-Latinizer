# Thaana Latinizer
Converts Thaana characters into latin

## Install
``` composer require thaana/thaana-latinizer ```

## Use
```php
$thaana = new ThaanaLatinizer();

$latinized = $thaana->latinize('މިއަދަކީ ރީތި ދުވަހެކެވެ.');
```
